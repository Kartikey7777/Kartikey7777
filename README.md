## 🛠️ Featured Projects

### 🕳️ PATROL — Real-Time AI Pothole Detection System
`Computer Vision` `Object Detection` `OpenCV` `Deep Learning`
- Engineered a CNN-based object detection pipeline (YOLOv5-style architecture) identifying potholes from live video with <50ms per-frame inference latency
- Applied OpenCV preprocessing (Gaussian blur, Canny edge detection, ROI masking) for robustness under varying lighting
- Implemented severity scoring (minor/moderate/severe) for road-maintenance prioritisation
- Outputs structured JSON alerts compatible with municipal GIS platforms for smart city integration

### 🌾 AgriBot — Intelligent AI Agriculture Chatbot
`NLP` `TensorFlow` `Intent Classification` `REST API`
- Built an intent-classification chatbot with TensorFlow + Keras achieving 85% accuracy on a custom agricultural corpus of 1,200+ labelled utterances
- Integrated OpenWeatherMap and soil-data APIs for dynamic, context-aware crop recommendations
- Constructed a full ML pipeline: data collection → tokenisation → embedding → training → evaluation → REST inference endpoint
- Added Hindi + English multilingual support to broaden accessibility for rural farming communities

### 🤟 AI Sign Language Interpreter — Real-Time Gesture-to-Speech
`OpenCV` `MediaPipe` `Pose Estimation` `LSTM`
- Developed a real-time gesture recognition system using MediaPipe Holistic (21 hand landmarks + pose), achieving 90% accuracy across 26 ASL signs
- Engineered a custom LSTM sequence classifier, reducing false-positive rate by ~30% vs. frame-level classification
- Implemented a text-to-speech output pipeline (pyttsx3) with <200ms end-to-end latency
- Optimised the inference graph for CPU-only deployment, sustaining 30 FPS on standard laptop hardware
